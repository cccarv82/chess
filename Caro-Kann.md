### QUICK REFERENCE:

```
REGRAS BÁSICAS:
✓ Bispo SEMPRE fora da cadeia
✓ Desenvolvimento antes de ataques
✓ Roque antes de ações
✓ c5 é jogada chave

ESTRUTURAS PRINCIPAIS:
TROCA (após exd5):
- Foco queenside
- Prepare b5-b4
- Ganhe peão c2

AVANÇADA (após e5):
- Ataque base com c5
- Pressione d4
- Prepare f6
```

### ARMADILHAS:

```
VS Nc3+Bf5: Ng3-h4-Ne4 → ganha bispo
VS e5: c5!-dxc5-Nc6-Qa5+ → ganha peça
VS Qf3: dxe4-Nf6-exf6 → vantagem

IMPORTANTE:
× NÃO bispo dentro da cadeia
× NÃO e6 antes de Bf5
× NÃO atacar sem desenvolvimento
```

### PLANOS POR VARIANTE:

```
TROCA:
- Foco TOTAL lado da dama
- Ignore lado do rei
- Prepare b5-b4
- Ganhe um peão e troque peças

AVANÇADA:
- Ataque base da cadeia (c5)
- Pressione d4
- Prepare f6
- Controle casas escuras

FANTASMA:
- Ataque o rei
- Peças ativas
- Explorar rei exposto
```

### ESTRUTURA BASE:

```
1. e4 c6    (Preparando d5)
2. ... d5    (Ataque ao centro)

IMPORTANTE:
- Bispo SEMPRE fora da cadeia
- Desenvolvimento natural
- Rei seguro primeiro
- Foco no lado da dama (não no rei!)
```

### VARIANTE DA TROCA (após 3.exd5):

```
1. e4 c6
2. ... d5
3. exd5 cxd5
4. ... Nf6
5. ... Bf5! (CRUCIAL - Bispo fora ANTES de e6!)
6. ... e6
7. ... Be7 (Mais seguro que Bd6)
8. ... O-O
9. ... Nc6
10. ... Qb6 (Melhor que Qc7)
11. ... Rfc8

PLANO PRINCIPAL:
- Foco TOTAL no lado da dama
- Ignore completamente o lado do rei
- Prepare b5-b4 para ataque minoria
- Objetivo: Ganhar um peão e trocar peças
```

### VARIANTE AVANÇADA (após 3.e5):

```
1. e4 c6
2. ... d5
3. e5 Bf5! (CRUCIAL - antes de e6!)
4. ... e6
5. ... c5 (Ataque na base!)
6. ... Nc6
7. ... Nge7 (Prepara f6)
8. ... Qb6

PLANO:
- Ataque base da cadeia (c5)
- Pressione d4
- Prepare f6
- Controle casas escuras

ALVO PRINCIPAL:
Peão d4 - não pode ser defendido por outros peões
```

### CONTRA ATAQUE FANTASMA (após 3.f3):

```
1. e4 c6
2. ... d5
3. f3 dxe4!
4. fxe4 e5!
5. ... Qh4+
6. ... Bc5
7. ... Nf6
8. ... O-O

PLANO:
- Ataque rei exposto
- Desenvolvimento rápido
- Pressão na diagonal a2-g8
```

### CONTRA Bc4 (NOVA VARIANTE):

```
1. e4 c6
2. ... d5
3. Bc4 dxe4
4. ... Nf6
5. ... Bg4!
6. ... e6
7. ... Bd6
8. ... O-O

IMPORTANTE:
- d6 URGENTE antes de Nf6
- Roque rápido
```

### CONTRA c3:

```
1. e4 c6
2. c3 (ou após Nf3 g6 d4 Bg7)
    d5! IMEDIATO

SE:
a) exd5: Nf6! (recupera peão)
b) e5: d4! (quebra centro)
c) d4: cxd4, cxd4, exd5
```

### IDEIAS ESTRATÉGICAS:

```
1. DESENVOLVIMENTO:
   - Bispo fora antes de e6
   - d6 antes de Nf6 (vs Bc4)
   - Roque antes de complicações

2. ATAQUES:
   - Lado da dama na variante troca
   - Centro na variante avançada
   - Rei na variante fantasma

3. PEÕES:
   - Ataque base da cadeia
   - Evite enfraquecer estrutura
   - Crie peões passados quando possível
```

### ERROS COMUNS A EVITAR:

```
1. NÃO:
   × Bispo dentro da cadeia
   × Trocar sem ganho claro
   × Atacar rei na variante troca

2. SIM:
   ✓ Desenvolvimento completo
   ✓ Foco no lado correto
   ✓ Roque antes de ataques
```

### TEMAS TÁTICOS:

```
1. VARIANTE TROCA:
   - Ataque minoria (b5-b4)
   - Pressão na coluna c
   - Peões isolados

2. VARIANTE AVANÇADA:
   - Ruptura com c5
   - Pressão em d4
   - Controle e4

3. VARIANTE FANTASMA:
   - Ataques de mate
   - Peças ativas
   - Rei exposto
```

### ARMADILHAS A EVITAR:

```
ERROS FATAIS:
× g5/g6 prematuros - enfraquecem rei
× Bf5 sem proteção - pode ser atacado
× Ataques sem desenvolvimento

SEQUÊNCIA SEGURA:
✓ c6-d5 primeiro
✓ Bispo fora com proteção
✓ Desenvolvimento completo
✓ Só então considerar ataques
```

### GUIA DE TROCAS:

```
QUANDO TROCAR:
✓ Troca igual com posição melhor
✓ Ganho material (usar tabela)
✓ Simplificação com vantagem

VALORES DAS PEÇAS:
• Peão = 1
• Cavalo/Bispo = 3
• Torre = 4.5
• Dama = 9

IMPORTANTE:
× NÃO trocar sem razão clara
× NÃO trocar se piorar estrutura
× NÃO trocar se ativar peças inimigas
```

### SEGURANÇA DA DAMA:

```
REGRAS CRÍTICAS:
✓ Dama atrás dos peões
✓ Desenvolvimento antes de ataques
✓ Evite exposição prematura

QUANDO MOVER A DAMA:
1. Para defender mate
2. Para capturar material
3. Para conectar torres
4. NUNCA sem razão clara
```
