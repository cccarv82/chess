### QUICK REFERENCE:

```
REGRAS BÁSICAS:
✓ Nc3 ANTES de Nf3 sempre!
✓ Bf4 antes de e3
✓ Ordem correta: Nc3 > Bf4 > Bd3 > Nf3
✓ Desenvolvimento antes de ataques

ESTRUTURAS PRINCIPAIS:
COLISEU (após ...Bxc3):
- Plano: c4-c5 duplo
- Par de bispos é vantagem

BOA CONSTRICTOR (após ...Bxf3):
- Peões duplos em f
- Controle de e4

GREGO (após e4-e5):
- Bispo em d3
- Bxh7+ quando possível
```

### RESPOSTAS RÁPIDAS:

```
VS BISPO DENTRO (e6):
1. d4 d5
2. Nc3 e6
3. Bf4 c5!
4. dxc5!
5. Bd3
6. e4!
7. e5

PLANO:
- Pressão com e4-e5
- Buscar sacrifício grego

VS BISPO FORA (Bf5):
1. d4 d5
2. Nc3 Bf5
3. f3!
4. g4 Bg6
5. h4
6. Nf3
7. Bd3

PLANO:
- Pawn storm no flanco rei
- Ataque no rei

VS ...g6:
1. d4 g6
2. Nc3 Nf6
3. e4 d6
4. Bf4 Bg7
5. e5! dxe5
6. dxe5 Qxd1+
7. Rxd1 Nfd7
8. Nd5!

PLANO:
- Forçar sacrifício de dama
- Mate em d8
```

### ARMADILHAS:

```
1. ARMADILHA DO MATE RÁPIDO (vs ...d6):
SEQUÊNCIA:
1. d4 d6
2. Nc3 Nf6
3. Bf4 g6
4. e4 Bg7
5. e5! dxe5
6. dxe5 Qxd1+
7. Rxd1 Nh5
8. Nd5! Nxf4
9. Nxc7+ Kf8
10. Rd8#

COMO IDENTIFICAR:
✓ Oponente joga ...d6 cedo
✓ Seguido de ...g6 e ...Bg7
✓ Padrão "fianchetto prematuro"

2. ARMADILHA DO CAVALO (vs ...c5):
SEQUÊNCIA:
1. d4 d5
2. Nc3 Nf6
3. Bf4 c5?
4. dxc5! b6?
5. Nb5! bxc5
6. Nc7+!

COMO IDENTIFICAR:
✓ Oponente joga ...c5 cedo
✓ Tenta atacar centro sem desenvolvimento
✓ Segue com ...b6 apressado

3. ARMADILHA DA DAMA (vs Bg4):
SEQUÊNCIA:
1. d4 Nf6
2. Nc3 d5
3. Bf4 c6
4. e3 Bg4?
5. f3! Bh5
6. g4! Bg6
7. h4! h6
8. Nxd5!

COMO IDENTIFICAR:
✓ Bispo preto sai cedo para g4
✓ Oponente não protege adequadamente
✓ Padrão "bispo exposto"

4. ARMADILHA DO FIANCHETTO (vs ...g6):
SEQUÊNCIA:
1. d4 g6
2. Nc3 Nf6
3. e4 d6
4. Bf4 Bg7
5. e5! dxe5
6. dxe5 Qxd1+
7. Rxd1 Nfd7
8. Nd5! Na6
9. Bxa6! bxa6
10. Nxc7+ Kf8
11. Nxa8 Nxe5
12. Rd8#

COMO IDENTIFICAR:
✓ Oponente faz fianchetto cedo
✓ Segue com ...d6 e ...Nf6
✓ Não protege adequadamente o rei
```
